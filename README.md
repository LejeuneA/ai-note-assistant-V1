# AI Note Assistant v1

AI Note Assistant v1 is a small rule-based Python command-line application that analyzes short notes and suggests a category, priority, and action.

This project was built as a beginner Python learning project to practice functions, lists, dictionaries, loops, conditionals, command handling, JSON save/load, and basic error handling.

## What the app does

The user can write short notes such as:

```text
Fix urgent login bug today
Create UI design this week
Prepare meeting notes soon
```

The app analyzes each note and returns:

```text
Category: Development / Design / Communication / General
Priority: High / Medium / Low
Action: Suggested next step
```

## Features

* Analyze short notes
* Detect note category using simple keyword rules
* Detect priority using simple keyword rules
* Suggest an action based on category and priority
* Show a session summary
* Count notes by priority
* Count notes by category
* Save analyzed notes to a JSON file
* Load saved notes from a JSON file
* Search notes by keyword
* Clear the current session
* Show help and rule explanations
* Handle missing or invalid JSON files safely

## Commands

| Command   | Shortcut | Description                      |
| --------- | -------- | -------------------------------- |
| `help`    | `h`      | Show available commands          |
| `rules`   | `r`      | Show category and priority rules |
| `summary` | `s`      | Show current session summary     |
| `save`    | `sv`     | Save notes to JSON               |
| `load`    | `ld`     | Load notes from JSON             |
| `clear`   | `c`      | Clear current session notes      |
| `search`  | `f`      | Search notes by keyword          |
| `quit`    | `q`      | Exit the app                     |

## Category rules

The app uses simple keyword matching:

```text
bug / fix        -> Development
design / ui      -> Design
meeting / call   -> Communication
otherwise        -> General
```

Note: The first matching category rule wins.

## Priority rules

```text
urgent / today                 -> High
this week / soon / tomorrow    -> Medium
otherwise                      -> Low
```

## Save and load notes

The app saves notes to:

```text
session_notes.json
```

This file is generated by the app and should not be committed to GitHub.

The `.gitignore` file should include:

```text
session_notes.json
session_notes.txt
```

## Search notes

The app can search notes that are currently loaded in the active session.

Example:

```text
search
Search keyword: bug
```

If saved notes exist in `session_notes.json`, load them first:

```text
load
search
```

## How to run the app

In the project folder, run:

```bash
python3 ai_note_assistant_V1.py
```

Or, depending on your Python setup:

```bash
python ai_note_assistant_V1.py
```
